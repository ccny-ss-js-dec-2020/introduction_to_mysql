## Sql Language
* Create Database
* Schema Change - Create Table
* Query - Create Records in Database
* Query - Read from Table
* Query - Where Clause
* Query - Join Clause
* Schema Change - Foreign Keys, Relationships
* Query - Deletes & Updates

## CRUD
* 4 CRUD Operations when calling a Database
  * Read == Get
  * Create == Post
  * Update == Put
  * Delete == Delete
